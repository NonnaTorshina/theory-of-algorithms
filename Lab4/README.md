# Лабораторная работа №4 - генераторы
## Вариант 20
## Описание
Приложение реализует три основные задачи варианта 20 лабораторной работы №4 с использованием функций-генераторов, многопоточности и графического интерфейса. Программа демонстрирует работу с генераторами Python, обработку исключений и повышение производительности через параллельные вычисления

## Задачи

- **Задание №1:** Имеется список из нот: 
p = ['до', 'ре', 'ми', 'фа', 'соль', 'ля', 'си']
Необходимо записать генератор, который бы случайным образом выдавал ноты из этого списка 1_000_000 раз. Для случайного выбора нот, используйте функцию choice модуля random. Выведите на экран первые 20 значений этого генератора.


- **Задание №2:** Определите функцию-генератор, которая бы возвращала целые числа кратные 3, начиная поиск от значения a, которое задается через параметр функции. Вызовите эту функции со значением аргумента a=-100 и выведите первые 20 чисел, возвращенных этой функцией-генератором. Какие-либо коллекции в программе не использовать.


- **Задание №3:** Вводится список email-адресов в одну строчку через пробел. Среди них нужно оставить только корректно записанные адреса. Будем полагать, что к таким относятся те, что используют латинские буквы, цифры и символ подчеркивания. А также в адресе сначала должен идти символ "@", а затем, точка "." (между ними, конечно же, могут быть и другие символы). Отобразить результат отбора корректных адресов на экране.


## Основные функции
- Генератор случайных музыкальных нот - создает последовательности из 7 основных нот

- Генератор чисел, кратных 3 - бесконечный генератор математической последовательности

- Валидатор email-адресов - проверяет корректность email по заданному шаблону

- Многопоточные версии генераторов - оптимизированные реализации для повышения производительности

## Инструкции по запуску
### Требования

- Python 3.8
- Установленные зависимости из requirements.txt

### Установка и запуск

1. Клонирование репозитория
```bash
git clone <repository-url>
cd Lab4
```

2. Установка зависимостей
```bash
pip install -r requirements.txt
```

3. Запуск приложения
```bash
python main.py
```

4. Запуск тестов
```bash
pytest tests.py -v
```

## Инструкция по работе с приложением
### Генератор нот
1. В поле "Количество нот" введите нужное число (от 1 до 1000)

2. Нажмите одну из кнопок:

3. "Обычный генератор" - базовая версия

4. "Многопоточные ноты" - ускоренная версия для больших объемов

5. Результат появится в текстовом поле ниже

Пример: При вводе 10 может выдать: "до ре ми соль ля фа до си ре соль"

### Числа, кратные 3
1. В поле "Начальное число" введите число, с которого начать поиск (например, -100)

2. В поле "Количество чисел" введите, сколько чисел сгенерировать

3. Нажмите "Сгенерировать числа"

4. Результат появится в текстовом поле

Пример: При начале с -100 и количестве 5: "-99 -96 -93 -90 -87"
### Валидация email

1. В поле ввода введите email-адреса через пробел

2. Нажмите "Проверить email"

3. В текстовом поле появятся только корректные адреса

4. **Формат корректного email:**

Только латинские буквы, цифры и символ подчеркивания

Обязательно содержит @ и точку после него

Примеры корректных: "test@mail.com", "user_name@domain.org"

Примеры некорректных: "invalid", "no@tld", "@nodomain.com"
